# 🚀 Zaptopay - High-Performance Crypto-to-XAF Exchange

Zaptopay is a premium B2B and peer-to-peer cryptocurrency exchange platform specifically optimized for the African market (Cameroon/CEMAC region). It provides real-time pricing, instant WhatsApp-integrated trades, and an institutional-grade administrative backend.

## ⚡ Features
- **Real-Time Settlement**: Public ledger with dynamic institutional price conversion.
- **WhatsApp Integration**: Hybrid automated & agent-led trade flows.
- **Admin Dashboard**: Real-time rate management and bot configuration.
- **Security**: Enforced Row Level Security (RLS) via Supabase.
